# Video

```
# Convert the video
python convert.py --file static/trailer-ticker.mp4
```
Another file **changed-trailer-ticker.mp4** should be created.
To scale the video I used **ffmpeg**.

Run:
-----
```
python server.py
```
See the video here: [http://localhost:8888](http://localhost:8888)

Requirements:
-----
```
tornado==4.1
```

Also ffmpeg.
